# apistatus-api

An API for [apistatus](https://github.com/Mashape/apistatus)

### Usage

#### Start the server

```
git clone https://github.com/Mashape/apistatus-api
cd apistatus-api
npm i
node server.js
```

#### Make a request

```
curl 127.0.0.1:8080/?url=https://mockbin.com/status/200
```
